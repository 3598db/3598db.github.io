# т╗║уФІУ┐ъТјЦСИЅТгАТЈАТЅІ

ждќтЁѕтЂЄУ«ЙСИ╗тіетЈЉУхиУ»иТ▒ѓуџёСИђуФ»уД░СИ║т«бТѕиуФ»№╝їУбФтіеУ┐ъТјЦуџёСИђуФ»уД░СИ║ТюЇтіАуФ»сђѓСИЇу«АТў»т«бТѕиуФ»У┐ўТў»ТюЇтіАуФ»№╝їTCP У┐ъТјЦт╗║уФІт«їтљјжЃйУЃйтЈЉжђЂтњїТјЦТћХТЋ░ТЇ«№╝їТЅђС╗Ц TCP Тў»СИђСИфтЁетЈїтиЦуџётЇЈУ««сђѓ

УхитѕЮ№╝їСИцуФ»жЃйСИ║ CLOSED уіХТђЂсђѓтюежђџС┐Ат╝ђтДІтЅЇ№╝їтЈїТќ╣жЃйС╝џтѕЏт╗║ TCBсђѓ ТюЇтіАтЎетѕЏт╗║т«ї TCB тљјСЙ┐У┐ЏтЁЦ LISTEN уіХТђЂ№╝їТГцТЌХт╝ђтДІуГЅтЙЁт«бТѕиуФ»тЈЉжђЂТЋ░ТЇ«сђѓ

# уггСИђТгАТЈАТЅІ

т«бТѕиуФ»тљЉТюЇтіАуФ»тЈЉжђЂУ┐ъТјЦУ»иТ▒ѓТіЦТќЄТ«хсђѓУ»ЦТіЦТќЄТ«хСИГтїЁтљФУЄфУ║ФуџёТЋ░ТЇ«жђџУ«»тѕЮтДІт║ЈтЈисђѓУ»иТ▒ѓтЈЉжђЂтљј№╝їт«бТѕиуФ»СЙ┐У┐ЏтЁЦ SYN-SENT уіХТђЂсђѓ

# уггС║їТгАТЈАТЅІ

ТюЇтіАуФ»ТћХтѕ░У┐ъТјЦУ»иТ▒ѓТіЦТќЄТ«хтљј№╝їтдѓТъютљїТёЈУ┐ъТјЦ№╝їтѕЎС╝џтЈЉжђЂСИђСИфт║ћуГћ№╝їУ»Цт║ћуГћСИГС╣ЪС╝џтїЁтљФУЄфУ║ФуџёТЋ░ТЇ«жђџУ«»тѕЮтДІт║ЈтЈи№╝їтЈЉжђЂт«їТѕљтљјСЙ┐У┐ЏтЁЦ SYN-RECEIVED уіХТђЂсђѓ

# уггСИЅТгАТЈАТЅІ

тйЊт«бТѕиуФ»ТћХтѕ░У┐ъТјЦтљїТёЈуџёт║ћуГћтљј№╝їУ┐ўУдЂтљЉТюЇтіАуФ»тЈЉжђЂСИђСИфуА«У«цТіЦТќЄсђѓт«бТѕиуФ»тЈЉт«їУ┐ЎСИфТіЦТќЄТ«хтљјСЙ┐У┐ЏтЁЦ ESTABLISHED уіХТђЂ№╝їТюЇтіАуФ»ТћХтѕ░У┐ЎСИфт║ћуГћтљјС╣ЪУ┐ЏтЁЦ ESTABLISHED уіХТђЂ№╝їТГцТЌХУ┐ъТјЦт╗║уФІТѕљтіЪсђѓ

PS№╝џуггСИЅТгАТЈАТЅІСИГтЈ»С╗ЦтїЁтљФТЋ░ТЇ«№╝їжђџУ┐Єт┐ФжђЪТЅЊт╝ђ№╝ѕTFO№╝ЅТіђТю»т░▒тЈ»С╗Цт«ъуј░У┐ЎСИђтіЪУЃйсђѓтЁХт«ътЈфУдЂТХЅтЈітѕ░ТЈАТЅІуџётЇЈУ««№╝їжЃйтЈ»С╗ЦСй┐ућеу▒╗С╝╝ TFO уџёТќ╣т╝Ј№╝їт«бТѕиуФ»тњїТюЇтіАуФ»тГўтѓеуЏИтљїуџё cookie№╝їСИІТгАТЈАТЅІТЌХтЈЉтЄ║ cookie УЙЙтѕ░тЄЈт░Љ RTT уџёуЏ«уџёсђѓ

тИИУђЃжЮбУ»Ћжбў№╝џСИ║С╗ђС╣ѕ TCP т╗║уФІУ┐ъТјЦжюђУдЂСИЅТгАТЈАТЅІ№╝їТўјТўјСИцТгАт░▒тЈ»С╗Цт╗║уФІУхиУ┐ъТјЦ

тЏаСИ║У┐ЎТў»СИ║С║єжў▓ТГбтЄ║уј░тц▒ТЋѕуџёУ┐ъТјЦУ»иТ▒ѓТіЦТќЄТ«хУбФТюЇтіАуФ»ТјЦТћХуџёТЃЁтєх№╝їС╗јУђїС║ДућЪжћЎУ»»сђѓ

тЈ»С╗ЦТЃ│У▒АтдѓСИІтю║ТЎ»сђѓт«бТѕиуФ»тЈЉжђЂС║єСИђСИфУ┐ъТјЦУ»иТ▒ѓ A№╝їСйєТў»тЏаСИ║уйЉу╗ютјЪтЏажђаТѕљС║єУХЁТЌХ№╝їУ┐ЎТЌХ TCP С╝џтљ»тіеУХЁТЌХжЄЇС╝ауџёТю║тѕХтєЇТгАтЈЉжђЂСИђСИфУ┐ъТјЦУ»иТ▒ѓ BсђѓТГцТЌХУ»иТ▒ѓжА║тѕЕтѕ░УЙЙТюЇтіАуФ»№╝їТюЇтіАуФ»т║ћуГћт«їт░▒т╗║уФІС║єУ»иТ▒ѓ№╝їуёХтљјТјЦТћХТЋ░ТЇ«тљјжЄіТћЙС║єУ┐ъТјЦсђѓ

тЂЄУ«ЙУ┐ЎТЌХтђЎУ┐ъТјЦУ»иТ▒ѓ A тюеСИцуФ»тЁ│жЌГтљју╗ѕС║јТіхУЙЙС║єТюЇтіАуФ»№╝їжѓБС╣ѕТГцТЌХТюЇтіАуФ»С╝џУ«цСИ║т«бТѕиуФ»тЈѕжюђУдЂт╗║уФІ TCP У┐ъТјЦ№╝їС╗јУђїт║ћуГћС║єУ»ЦУ»иТ▒ѓт╣ХУ┐ЏтЁЦ ESTABLISHED уіХТђЂсђѓСйєТў»т«бТѕиуФ»тЁХт«ъТў» CLOSED уџёуіХТђЂ№╝їжѓБС╣ѕт░▒С╝џт»╝УЄ┤ТюЇтіАуФ»СИђуЏ┤уГЅтЙЁ№╝їжђаТѕљУхёТ║љуџёТхфУ┤╣сђѓ

PS№╝џтюет╗║уФІУ┐ъТјЦСИГ№╝їС╗╗ТёЈСИђуФ»ТјЅу║┐№╝їTCP жЃйС╝џжЄЇтЈЉ SYN тїЁ№╝їСИђУѕгС╝џжЄЇУ»ЋС║ћТгА№╝їтюет╗║уФІУ┐ъТјЦСИГтЈ»УЃйС╝џжЂЄтѕ░ SYN Flood Тћ╗тЄ╗сђѓжЂЄтѕ░У┐ЎуДЇТЃЁтєхСйатЈ»С╗ЦжђЅТІЕУ░ЃСйјжЄЇУ»ЋТгАТЋ░ТѕќУђЁт╣▓УёєтюеСИЇУЃйтцёуљєуџёТЃЁтєхСИІТІњу╗ЮУ»иТ▒ѓсђѓ

# ТќГт╝ђжЊЙТјЦтЏЏТгАТЈАТЅІ

УІЦт«бТѕиуФ» A У«цСИ║ТЋ░ТЇ«тЈЉжђЂт«їТѕљ№╝їтѕЎт«ЃжюђУдЂтљЉТюЇтіАуФ» B тЈЉжђЂУ┐ъТјЦжЄіТћЙУ»иТ▒ѓсђѓ

# уггС║їТгАТЈАТЅІ

B ТћХтѕ░У┐ъТјЦжЄіТћЙУ»иТ▒ѓтљј№╝їС╝џтЉіУ»Ѕт║ћућет▒ѓУдЂжЄіТћЙ TCP жЊЙТјЦсђѓуёХтљјС╝џтЈЉжђЂ ACK тїЁ№╝їт╣ХУ┐ЏтЁЦ CLOSE_WAIT уіХТђЂ№╝їТГцТЌХУАеТўј A тѕ░ B уџёУ┐ъТјЦти▓у╗ЈжЄіТћЙ№╝їСИЇтєЇТјЦТћХ A тЈЉуџёТЋ░ТЇ«С║єсђѓСйєТў»тЏаСИ║ TCP У┐ъТјЦТў»тЈїтљЉуџё№╝їТЅђС╗Ц B С╗ЇТЌДтЈ»С╗ЦтЈЉжђЂТЋ░ТЇ«у╗Ў Aсђѓ

# уггСИЅТгАТЈАТЅІ

B тдѓТъюТГцТЌХУ┐ўТюЅТ▓АтЈЉт«їуџёТЋ░ТЇ«С╝џу╗Ду╗ГтЈЉжђЂ№╝їт«їТ»ЋтљјС╝џтљЉ A тЈЉжђЂУ┐ъТјЦжЄіТћЙУ»иТ▒ѓ№╝їуёХтљј B СЙ┐У┐ЏтЁЦ LAST-ACK уіХТђЂсђѓ

PS№╝џжђџУ┐Єт╗ХУ┐ЪуА«У«цуџёТіђТю»№╝ѕжђџтИИТюЅТЌХжЌ┤жЎљтѕХ№╝їтљдтѕЎт»╣Тќ╣С╝џУ»»У«цСИ║жюђУдЂжЄЇС╝а№╝Ѕ№╝їтЈ»С╗Цт░єуггС║їТгАтњїуггСИЅТгАТЈАТЅІтљѕт╣Х№╝їт╗ХУ┐Ъ ACK тїЁуџётЈЉжђЂсђѓ

# уггтЏЏТгАТЈАТЅІ

A ТћХтѕ░жЄіТћЙУ»иТ▒ѓтљј№╝їтљЉ B тЈЉжђЂуА«У«цт║ћуГћ№╝їТГцТЌХ A У┐ЏтЁЦ TIME-WAIT уіХТђЂсђѓУ»ЦуіХТђЂС╝џТїЂу╗Г 2MSL№╝ѕТюђтцДТ«хућЪтГўТюЪ№╝їТїЄТіЦТќЄТ«хтюеуйЉу╗юСИГућЪтГўуџёТЌХжЌ┤№╝їУХЁТЌХС╝џУбФТіЏт╝Ѓ№╝Ѕ ТЌХжЌ┤№╝їУІЦУ»ЦТЌХжЌ┤Т«хтєЁТ▓АТюЅ B уџёжЄЇтЈЉУ»иТ▒ѓуџёУ»Ю№╝їт░▒У┐ЏтЁЦ CLOSED уіХТђЂсђѓтйЊ B ТћХтѕ░уА«У«цт║ћуГћтљј№╝їС╣ЪСЙ┐У┐ЏтЁЦ CLOSED уіХТђЂсђѓ

СИ║С╗ђС╣ѕ A УдЂУ┐ЏтЁЦ TIME-WAIT уіХТђЂ№╝їуГЅтЙЁ 2MSL ТЌХжЌ┤тљјТЅЇУ┐ЏтЁЦ CLOSED уіХТђЂ№╝Ъ

СИ║С║єС┐ЮУ»Ђ B УЃйТћХтѕ░ A уџёуА«У«цт║ћуГћсђѓУІЦ A тЈЉт«їуА«У«цт║ћуГћтљјуЏ┤ТјЦУ┐ЏтЁЦ CLOSED уіХТђЂ№╝їтдѓТъюуА«У«цт║ћуГћтЏаСИ║уйЉу╗южЌ«жбўСИђуЏ┤Т▓АТюЅтѕ░УЙЙ№╝їжѓБС╣ѕС╝џжђаТѕљ B СИЇУЃйТГБтИИтЁ│жЌГсђѓ

# ARQ тЇЈУ««

ARQ тЇЈУ««С╣Ът░▒Тў»УХЁТЌХжЄЇС╝аТю║тѕХсђѓжђџУ┐ЄуА«У«цтњїУХЁТЌХТю║тѕХС┐ЮУ»ЂС║єТЋ░ТЇ«уџёТГБуА«жђЂУЙЙ№╝їARQ тЇЈУ««тїЁтљФтЂюТГбуГЅтЙЁ ARQ тњїУ┐ъу╗Г ARQ СИцуДЇтЇЈУ««сђѓ

# ТГБтИИС╝аУЙЊУ┐ЄуеІ

# ТіЦТќЄСИбтц▒ТѕќтЄ║жћЎ

тюеТіЦТќЄС╝аУЙЊуџёУ┐ЄуеІСИГтЈ»УЃйС╝џтЄ║уј░СИбтїЁсђѓУ┐ЎТЌХтђЎУХЁУ┐Єт«џТЌХтЎеУ«Йт«џуџёТЌХжЌ┤т░▒С╝џтєЇТгАтЈЉжђЂСИбтц▒уџёТЋ░ТЇ«уЏ┤тѕ░т»╣уФ»тЊЇт║ћ№╝їТЅђС╗ЦжюђУдЂТ»ЈТгАжЃйтцЄС╗йтЈЉжђЂуџёТЋ░ТЇ«сђѓ

тЇ│Сй┐ТіЦТќЄТГБтИИуџёС╝аУЙЊтѕ░т»╣уФ»№╝їС╣ЪтЈ»УЃйтЄ║уј░тюеС╝аУЙЊУ┐ЄуеІСИГТіЦТќЄтЄ║жћЎуџёжЌ«жбўсђѓУ┐ЎТЌХтђЎт»╣уФ»С╝џТіЏт╝ЃУ»ЦТіЦТќЄт╣ХуГЅтЙЁ A уФ»жЄЇС╝асђѓ

PS№╝џСИђУѕгт«џТЌХтЎеУ«Йт«џуџёТЌХжЌ┤жЃйС╝џтцДС║јСИђСИф RTT уџёт╣│тЮЄТЌХжЌ┤сђѓ

# ACK УХЁТЌХТѕќСИбтц▒

т»╣уФ»С╝аУЙЊуџёт║ћуГћС╣ЪтЈ»УЃйтЄ║уј░СИбтц▒ТѕќУХЁТЌХуџёТЃЁтєхсђѓжѓБС╣ѕУХЁУ┐Єт«џТЌХтЎеТЌХжЌ┤ A уФ»уЁДТаиС╝џжЄЇС╝аТіЦТќЄсђѓУ┐ЎТЌХтђЎ B уФ»ТћХтѕ░уЏИтљїт║ЈтЈиуџёТіЦТќЄС╝џСИбт╝ЃУ»ЦТіЦТќЄт╣ХжЄЇС╝ат║ћуГћ№╝їуЏ┤тѕ░ A уФ»тЈЉжђЂСИІСИђСИфт║ЈтЈиуџёТіЦТќЄсђѓ

тюеУХЁТЌХуџёТЃЁтєхСИІС╣ЪтЈ»УЃйтЄ║уј░т║ћуГћтЙѕУ┐Ътѕ░УЙЙ№╝їУ┐ЎТЌХ A уФ»С╝џтѕцТќГУ»Цт║ЈтЈиТў»тљдти▓у╗ЈТјЦТћХУ┐Є№╝їтдѓТъюТјЦТћХУ┐ЄтЈфжюђУдЂСИбт╝Ѓт║ћуГћтЇ│тЈ»сђѓ

С╗јСИіжЮбуџёТЈЈУ┐░СИГтцДт«ХУѓ»т«џтЈ»С╗ЦтЈЉуј░У┐ЎУѓ»т«џСИЇТў»СИђСИфжФўТЋѕуџёТќ╣т╝ЈсђѓтЂЄУ«ЙтюеУЅ»тЦйуџёуйЉу╗юуј»тбЃСИГ№╝їТ»ЈТгАтЈЉжђЂТЋ░ТЇ«жЃйжюђУдЂуГЅтЙЁуЅЄтѕ╗Уѓ»т«џТў»СИЇУЃйТјЦтЈЌуџёсђѓжѓБС╣ѕТЌбуёХТѕЉС╗гСИЇУЃйТјЦтЈЌУ┐ЎСИфСИЇжѓБС╣ѕжФўТЋѕуџётЇЈУ««№╝їт░▒ТЮЦу╗Ду╗ГтГдС╣ауЏИт»╣жФўТЋѕуџётЇЈУ««тљДсђѓ

# У┐ъу╗Г ARQ

тюеУ┐ъу╗Г ARQ СИГ№╝їтЈЉжђЂуФ»ТІЦТюЅСИђСИфтЈЉжђЂуфЌтЈБ№╝їтЈ»С╗ЦтюеТ▓АТюЅТћХтѕ░т║ћуГћуџёТЃЁтєхСИІТїЂу╗ГтЈЉжђЂуфЌтЈБтєЁуџёТЋ░ТЇ«№╝їУ┐ЎТаиуЏИТ»ћтЂюТГбуГЅтЙЁ ARQ тЇЈУ««ТЮЦУ»┤тЄЈт░ЉС║єуГЅтЙЁТЌХжЌ┤№╝їТЈљжФўС║єТЋѕујЄсђѓ

# у┤»У«АуА«У«ц

У┐ъу╗Г ARQ СИГ№╝їТјЦТћХуФ»С╝џТїЂу╗ГСИЇТќГТћХтѕ░ТіЦТќЄсђѓтдѓТъютњїтЂюТГбуГЅтЙЁ ARQ СИГТјЦТћХСИђСИфТіЦТќЄт░▒тЈЉжђЂСИђСИфт║ћуГћСИђТаи№╝їт░▒тцфТхфУ┤╣УхёТ║љС║єсђѓжђџУ┐Єу┤»У«АуА«У«ц№╝їтЈ»С╗ЦтюеТћХтѕ░тцџСИфТіЦТќЄС╗Цтљју╗ЪСИђтЏътцЇСИђСИфт║ћуГћТіЦТќЄсђѓТіЦТќЄСИГуџё ACK ТаЄт┐ЌСйЇтЈ»С╗ЦућеТЮЦтЉіУ»ЅтЈЉжђЂуФ»У┐ЎСИфт║ЈтЈиС╣ІтЅЇуџёТЋ░ТЇ«ти▓у╗ЈтЁежЃеТјЦТћХтѕ░С║є№╝їСИІТгАУ»итЈЉжђЂУ┐ЎСИфт║ЈтЈитљјуџёТЋ░ТЇ«сђѓ

СйєТў»у┤»У«АуА«У«цС╣ЪТюЅСИђСИфт╝іуФ»сђѓтюеУ┐ъу╗ГТјЦТћХТіЦТќЄТЌХ№╝їтЈ»УЃйС╝џжЂЄтѕ░ТјЦТћХтѕ░т║ЈтЈи 5 уџёТіЦТќЄтљј№╝їт╣ХТюфТјЦТћХтѕ░т║ЈтЈи 6 уџёТіЦТќЄ№╝їуёХУђїт║ЈтЈи 7 С╗ЦтљјуџёТіЦТќЄти▓у╗ЈТјЦТћХсђѓжЂЄтѕ░У┐ЎуДЇТЃЁтєхТЌХ№╝їACK тЈфУЃйтЏътцЇ 6№╝їУ┐ЎТаит░▒С╝џжђаТѕљтЈЉжђЂуФ»жЄЇтцЇтЈЉжђЂТЋ░ТЇ«уџёТЃЁтєхсђѓ

# Т╗ЉтіеуфЌтЈБ

тюеСИіжЮбт░ЈУіѓСИГУ«▓тѕ░С║єтЈЉжђЂуфЌтЈБсђѓтюе TCP СИГ№╝їСИцуФ»тЁХт«ъжЃйу╗┤ТіцуЮђуфЌтЈБ№╝џтѕєтѕФСИ║тЈЉжђЂуФ»уфЌтЈБтњїТјЦТћХуФ»уфЌтЈБсђѓ

тЈЉжђЂуФ»уфЌтЈБтїЁтљФти▓тЈЉжђЂСйєТюфТћХтѕ░т║ћуГћуџёТЋ░ТЇ«тњїтЈ»С╗ЦтЈЉжђЂСйєТў»ТюфтЈЉжђЂуџёТЋ░ТЇ«сђѓ

тЈЉжђЂуФ»уфЌтЈБТў»ућ▒ТјЦТћХуфЌтЈБтЅЕСйЎтцДт░Јтє│т«џуџёсђѓТјЦТћХТќ╣С╝џТіітйЊтЅЇТјЦТћХуфЌтЈБуџётЅЕСйЎтцДт░ЈтєЎтЁЦт║ћуГћТіЦТќЄ№╝їтЈЉжђЂуФ»ТћХтѕ░т║ћуГћтљјТа╣ТЇ«У»Цтђ╝тњїтйЊтЅЇуйЉу╗юТІЦтАъТЃЁтєхУ«Йуй«тЈЉжђЂуфЌтЈБуџётцДт░Ј№╝їТЅђС╗ЦтЈЉжђЂуфЌтЈБуџётцДт░ЈТў»СИЇТќГтЈўтїќуџёсђѓ

тйЊтЈЉжђЂуФ»ТјЦТћХтѕ░т║ћуГћТіЦТќЄтљј№╝їС╝џжџЈС╣Іт░єуфЌтЈБУ┐ЏУАїТ╗Љтіе

Т╗ЉтіеуфЌтЈБТў»СИђСИфтЙѕжЄЇУдЂуџёТдѓт┐х№╝їт«ЃтИ«тіЕ TCP т«ъуј░С║єТхЂжЄЈТјДтѕХуџётіЪУЃйсђѓТјЦТћХТќ╣жђџУ┐ЄТіЦТќЄтЉіуЪЦтЈЉжђЂТќ╣У┐ўтЈ»С╗ЦтЈЉжђЂтцџт░ЉТЋ░ТЇ«№╝їС╗јУђїС┐ЮУ»ЂТјЦТћХТќ╣УЃйтцЪТЮЦтЙЌтЈіТјЦТћХТЋ░ТЇ«№╝їжў▓ТГбтЄ║уј░ТјЦТћХТќ╣тИдт«йти▓Т╗А№╝їСйєТў»тЈЉжђЂТќ╣У┐ўСИђуЏ┤тЈЉжђЂТЋ░ТЇ«уџёТЃЁтєхсђѓ

# Zero уфЌтЈБ

тюетЈЉжђЂТіЦТќЄуџёУ┐ЄуеІСИГ№╝їтЈ»УЃйС╝џжЂЄтѕ░т»╣уФ»тЄ║уј░жЏХуфЌтЈБуџёТЃЁтєхсђѓтюеУ»ЦТЃЁтєхСИІ№╝їтЈЉжђЂуФ»С╝џтЂюТГбтЈЉжђЂТЋ░ТЇ«№╝їт╣Хтљ»тіе persistent timer сђѓУ»Цт«џТЌХтЎеС╝џт«џТЌХтЈЉжђЂУ»иТ▒ѓу╗Ўт»╣уФ»№╝їУ«Ет»╣уФ»тЉіуЪЦуфЌтЈБтцДт░ЈсђѓтюежЄЇУ»ЋТгАТЋ░УХЁУ┐ЄСИђт«џТгАТЋ░тљј№╝їтЈ»УЃйС╝џСИГТќГ TCP жЊЙТјЦсђѓ

# ТІЦтАътцёуљє

ТІЦтАътцёуљєтњїТхЂжЄЈТјДтѕХСИЇтљї№╝їтљјУђЁТў»СйюућеС║јТјЦТћХТќ╣№╝їС┐ЮУ»ЂТјЦТћХТќ╣ТЮЦтЙЌтЈіТјЦтЈЌТЋ░ТЇ«сђѓУђїтЅЇУђЁТў»СйюућеС║јуйЉу╗ю№╝їжў▓ТГбУ┐ЄтцџуџёТЋ░ТЇ«ТІЦтАъуйЉу╗ю№╝їжЂ┐тЁЇтЄ║уј░уйЉу╗юУ┤ЪУййУ┐ЄтцДуџёТЃЁтєхсђѓ

ТІЦтАътцёуљєтїЁТІгС║єтЏЏСИфу«ЌТ│Ћ№╝їтѕєтѕФСИ║№╝џТЁбт╝ђтДІ№╝їТІЦтАъжЂ┐тЁЇ№╝їт┐ФжђЪжЄЇС╝а№╝їт┐ФжђЪТЂбтцЇсђѓ

# ТЁбт╝ђтДІу«ЌТ│Ћ

ТЁбт╝ђтДІу«ЌТ│Ћ№╝їжАЙтљЇТђЮС╣Ѕ№╝їт░▒Тў»тюеС╝аУЙЊт╝ђтДІТЌХт░єтЈЉжђЂуфЌтЈБТЁбТЁбТїЄТЋ░у║ДТЅЕтцД№╝їС╗јУђїжЂ┐тЁЇСИђт╝ђтДІт░▒С╝аУЙЊтцДжЄЈТЋ░ТЇ«т»╝УЄ┤уйЉу╗юТІЦтАъсђѓТЃ│т┐ЁтцДт«ХжЃйСИІУййУ┐ЄУхёТ║љ№╝їТ»ЈтйЊТѕЉС╗гт╝ђтДІСИІУййуџёТЌХтђЎжЃйС╝џтЈЉуј░СИІУййжђЪт║дТў»ТЁбТЁбТЈљтЇЄуџё№╝їУђїСИЇТў»СИђУ╣┤Уђїт░▒уЏ┤ТјЦТІЅТ╗АтИдт«йсђѓ

ТЁбт╝ђтДІу«ЌТ│ЋТГЦжфцтЁиСйЊтдѓСИІ

  1. У┐ъТјЦтѕЮтДІУ«Йуй«ТІЦтАъуфЌтЈБ№╝ѕCongestion Window№╝Ѕ СИ║ 1 MSS№╝ѕСИђСИфтѕєТ«хуџёТюђтцДТЋ░ТЇ«жЄЈ№╝Ѕ
  2. Т»ЈУ┐ЄСИђСИф RTT т░▒т░єуфЌтЈБтцДт░ЈС╣ўС║ї
  3. ТїЄТЋ░у║ДтбъжЋ┐Уѓ»т«џСИЇУЃйТ▓АТюЅжЎљтѕХуџё№╝їТЅђС╗ЦТюЅСИђСИфжўѕтђ╝жЎљтѕХ№╝їтйЊуфЌтЈБтцДт░ЈтцДС║јжўѕтђ╝ТЌХт░▒С╝џтљ»тіеТІЦтАъжЂ┐тЁЇу«ЌТ│Ћсђѓ

# ТІЦтАъжЂ┐тЁЇу«ЌТ│Ћ

ТІЦтАъжЂ┐тЁЇу«ЌТ│ЋуЏИТ»ћу«ђтЇЋуѓ╣№╝їТ»ЈУ┐ЄСИђСИф RTT уфЌтЈБтцДт░ЈтЈфтіаСИђ№╝їУ┐ЎТаиУЃйтцЪжЂ┐тЁЇТїЄТЋ░у║ДтбъжЋ┐т»╝УЄ┤уйЉу╗юТІЦтАъ№╝їТЁбТЁбт░єтцДт░ЈУ░ЃТЋ┤тѕ░ТюђСй│тђ╝сђѓ

тюеС╝аУЙЊУ┐ЄуеІСИГтЈ»УЃйт«џТЌХтЎеУХЁТЌХуџёТЃЁтєх№╝їУ┐ЎТЌХтђЎ TCP С╝џУ«цСИ║уйЉу╗юТІЦтАъС║є№╝їС╝џжЕгСИіУ┐ЏУАїС╗ЦСИІТГЦжфц№╝џ

  1. т░єжўѕтђ╝У«ЙСИ║тйЊтЅЇТІЦтАъуфЌтЈБуџёСИђтЇі
  2. т░єТІЦтАъуфЌтЈБУ«ЙСИ║ 1 MSS
  3. тљ»тіеТІЦтАъжЂ┐тЁЇу«ЌТ│Ћ

# т┐ФжђЪжЄЇС╝а

т┐ФжђЪжЄЇС╝аСИђУѕгтњїт┐ФТЂбтцЇСИђУхитЄ║уј░сђѓСИђТЌдТјЦТћХуФ»ТћХтѕ░уџёТіЦТќЄтЄ║уј░тц▒т║ЈуџёТЃЁтєх№╝їТјЦТћХуФ»тЈфС╝џтЏътцЇТюђтљјСИђСИфжА║т║ЈТГБуА«уџёТіЦТќЄт║ЈтЈисђѓтдѓТъютЈЉжђЂуФ»ТћХтѕ░СИЅСИфжЄЇтцЇуџё ACK№╝їТЌажюђуГЅтЙЁт«џТЌХтЎеУХЁТЌХУђїТў»уЏ┤ТјЦтљ»тіет┐ФжђЪжЄЇС╝ау«ЌТ│ЋсђѓтЁиСйЊу«ЌТ│ЋтѕєСИ║СИцуДЇ№╝џ

# TCP Taho т«ъуј░тдѓСИІ

  • т░єжўѕтђ╝У«ЙСИ║тйЊтЅЇТІЦтАъуфЌтЈБуџёСИђтЇі
  • т░єТІЦтАъуфЌтЈБУ«ЙСИ║ 1 MSS
  • жЄЇТќ░т╝ђтДІТЁбт╝ђтДІу«ЌТ│Ћ

# TCP Reno т«ъуј░тдѓСИІ

  • ТІЦтАъуфЌтЈБтЄЈтЇі
  • т░єжўѕтђ╝У«ЙСИ║тйЊтЅЇТІЦтАъуфЌтЈБ
  • У┐ЏтЁЦт┐ФТЂбтцЇжўХТ«х№╝ѕжЄЇтЈЉт»╣уФ»жюђУдЂуџётїЁ№╝їСИђТЌдТћХтѕ░СИђСИфТќ░уџё ACK уГћтцЇт░▒жђђтЄ║У»ЦжўХТ«х№╝Ѕ№╝їУ┐ЎуДЇТќ╣т╝ЈтюеСИбтц▒тцџСИфтїЁуџёТЃЁтєхСИІт░▒СИЇжѓБС╣ѕтЦйС║є
  • Сй┐ућеТІЦтАъжЂ┐тЁЇу«ЌТ│Ћ

# TCP New Ren Тћ╣У┐Џтљјуџёт┐ФТЂбтцЇ

TCP New Reno у«ЌТ│ЋТћ╣У┐ЏС║єС╣ІтЅЇ TCP Reno у«ЌТ│Ћуџёу╝║жЎисђѓтюеС╣ІтЅЇ№╝їт┐ФТЂбтцЇСИГтЈфУдЂТћХтѕ░СИђСИфТќ░уџё ACK тїЁ№╝їт░▒С╝џжђђтЄ║т┐ФТЂбтцЇсђѓ

тюе TCP New Reno СИГ№╝їTCP тЈЉжђЂТќ╣тЁѕУ«░СИІСИЅСИфжЄЇтцЇ ACK уџётѕєТ«хуџёТюђтцДт║ЈтЈисђѓ

тЂЄтдѓТѕЉТюЅСИђСИфтѕєТ«хТЋ░ТЇ«Тў» 1 ~ 10 У┐ЎтЇЂСИфт║ЈтЈиуџёТіЦТќЄ№╝їтЁХСИГСИбтц▒С║єт║ЈтЈиСИ║ 3 тњї 7 уџёТіЦТќЄ№╝їжѓБС╣ѕУ»ЦтѕєТ«хуџёТюђтцДт║ЈтЈит░▒Тў» 10сђѓтЈЉжђЂуФ»тЈфС╝џТћХтѕ░ ACK т║ЈтЈиСИ║ 3 уџёт║ћуГћсђѓУ┐ЎТЌХтђЎжЄЇтЈЉт║ЈтЈиСИ║ 3 уџёТіЦТќЄ№╝їТјЦТћХТќ╣жА║тѕЕТјЦТћХуџёУ»Ют░▒С╝џтЈЉжђЂ ACK т║ЈтЈиСИ║ 7 уџёт║ћуГћсђѓУ┐ЎТЌХтђЎ TCP уЪЦжЂЊт»╣уФ»Тў»ТюЅтцџСИфтїЁТюфТћХтѕ░№╝їС╝џу╗Ду╗ГтЈЉжђЂт║ЈтЈиСИ║ 7 уџёТіЦТќЄ№╝їТјЦТћХТќ╣жА║тѕЕТјЦТћХт╣ХС╝џтЈЉжђЂ ACK т║ЈтЈиСИ║ 11 уџёт║ћуГћ№╝їУ┐ЎТЌХтЈЉжђЂуФ»У«цСИ║У┐ЎСИфтѕєТ«хТјЦТћХуФ»ти▓у╗ЈжА║тѕЕТјЦТћХ№╝їТјЦСИІТЮЦС╝џжђђтЄ║т┐ФТЂбтцЇжўХТ«хсђѓ

# т░Ју╗Њ

У┐ЎСИђуФаУіѓтєЁт«╣тЙѕтцџ№╝їтЁЁТќЦС║єтцДжЄЈуџёТю»У»Г№╝їжђѓтљѕтцДт«ХтЈЇтцЇуаћУ»╗№╝їти▓у╗ЈТіі TCP СИГТюђТаИт┐ЃТюђжюђУдЂТјїТЈАуџётєЁт«╣тЁеуЏўТЅўтЄ║С║є№╝їтдѓТюЅтЊфжЄїСИЇТўјуЎйуџёТгбУ┐јТЈљжЌ«сђѓ
